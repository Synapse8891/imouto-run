# Change Log

## [0.0.1] - 2025-12-09

### Added
- Initial release
- Support for Python (.py) file execution and error checking
- Support for C (.c) file compilation checking with gcc
- Support for Shell/Bash/Zsh (.sh, .bash, .zsh) syntax checking
- Imouto character translation of error messages
- Automatic line number extraction from errors
- Configurable output verbosity levels (simple, normal, verbose)
- Auto-run on startup, file open, or save
- Customizable translation rules for different error types

### Features
- Execute Python files and display errors
- Compile C files and report syntax errors
- Check shell script syntax
- Translate technical error messages into cute imouto character speech
- Multiple verbosity levels for different needs
- Extensible translation dictionary
